# 📊 Bank Customer Churn Analysis Dashboard (Power BI)

🎯 Objective

The objective of this project is to analyze bank customer churn patterns and identify the key factors influencing customer retention. Using interactive data visualizations, the dashboard helps uncover trends related to customer demographics, account balance, and activity status to support data-driven decision-making.

🚀 Project Overview

This project presents an interactive Power BI dashboard that analyzes customer churn using important banking metrics. It converts raw customer data into clear visual insights that help understand why customers leave the bank.

📌 Key Insights

👥 Total Customers: 10K
📉 Churn Customers: 2K
📊 Churn Rate: 20%
💳 Average Credit Score: 651
💰 Average Balance: $76K

🛠 Tools & Technologies

📊 Power BI Desktop – Data visualization and dashboard development

📈 Conclusion

The analysis highlights that customer activity level, account balance, and demographic factors play an important role in customer churn. By identifying these patterns, banks can develop targeted retention strategies, improve customer engagement, and reduce churn rates through data-driven decision making.
